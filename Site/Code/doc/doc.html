Partie I
Gérer la base de donnée

Connectez vous au panneau d’administration. Quand Denis se sera un peu bougé, vous devriez devoir entrer un mot de passe. Utiliser “baleine” comme identifiant et “phoque” comme mot de passe.

0.1  Ajouter un élément

0.1.1  Une corbeille

Séléctionnez “Corbeille” dans la liste déroulante. Ajouter une corbeille en renseignant son nom dans le champ prévu à cet effet.

0.1.2  Une sonde

Séléctionnez “Sonde” dans la liste déroulante. Trois champs s’afficheront, remplissez les tous avec les bonnes valeurs.

Appartient à
L’id de la corbeille à laquelle la sonde est rattachée.
Nom de la sonde
Le nom de la sonde.
Niveau
Le niveau de profondeur de la sonde.

0.1.3  Un puit

Séléctionnez “Puits” dans la liste déroulante. Remplissez le champs avec le nom du nouveau puit.
