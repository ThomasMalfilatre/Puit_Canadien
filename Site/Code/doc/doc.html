    Partie I
    Gérer les graphiques

    Pour accéder au graphique, il suffit de cliquer sur "Graphique".

    1.1.  Compréhension du graphique

    1.1.1. Sélectionner une sonde

    Pour sélectionner une sonde, il suffit de cliquer avec le clic GAUCHE ou DROIT de la souris sur la sonde voulue.

    1.1.2. Désélectionner une sonde

    Pour déselectionner une sonde, il suffit de cliquer avec le clic GAUCHE ou DROIT de la souris sur la sonde.

    1.1.3. Selectionner les dates

    Cliquer sur la zone texte réservée à cet effet pour ouvrir un calendrier et sélectionner la date.
    ATTENTION, le format s'affichera en MM/JJ/AAAA

    1.1.4  Générer le graphe température

    Une fois, toutes les sondes voulues sélectionnées ainsi que les dates de début et de fin il suffit de cliquer sur le bouton "Générer graphe" pour afficher le graphe des températures.


    1.2. Graphe des températures

    Une fois le bouton "Générer graphe" cliqué, le graphe des températures va apparaitre avec une courbe de couleur différente pour chaque sonde. La température selon l'axe des ordonnées et les différentes dates suivant l'axe des abscisses.

    1.2.1. Le Zoom

    Pour zoomer sur une partie du graphe il faut maintenir la souris appuyé du début de la zone à zoomer et, tout en mainteant la souris maintenue, décaler la souris jusqu'à la fin de la zone à zoomer. Enfin, il faut relâcher la souris et le graphe va zoomer automatiquement sur cette partie.
    Pour remettre le zoom à 0, il faut cliquer sur "Reset Zoom" situé en haut à droite.

    1.2.2. L'exportation du graphe

    Le graphe des températures peut s'exporter de différentes façons :

    • En format PNG
    • En format JPEG
    • En format PDF
    • En format SVG
    Pour cela, il faut cliquer sur le menu situé en haut au droite du graphe.

    1.2.3. L'impression

    L'impression du graphe est aussi disponible. Pour cela, il suffit d'aller dans le menu situé en haut à droite et de cliquer sur "Print Chart". La fenêtre d'impression va alors s'ouvrir.


    1.3. Position des sondes

    L'onglet "Position des sondes" sert à afficher le placement des sondes grâce à un schéma en 2D permettant de se repérer, en plus du 3D.



    Partie II
    Administration

    Pour accéder à la partie Administration de la base de données il faut cliquer sur Administration et entrer le mot de passe.

    2.1. Ajout de données

    2.1.1. Ajouter une sonde

    Pour ajouter une sonde, il faut aller dans le menu "Ajouter une sonde".
    Ensuite, il faut dire à quelle corbeille cette sonde appartient, le nom donné à la sonde dans la base de données et sur le modèle 3D. Enfin, il faut préciser ses positions, pour la représentation sur le modèle : x pour la largeur, y pour la longueur et z pour la profondeur.

    2.1.2. Ajouter une corbeille

    Pour ajouter une corbeille, il faut aller dans le menu "Ajouter une corbeille".
    Ensuite, il suffit de donner le nom de la corbeille et également les positions, pour la représentation sur le modèle : x pour la largeur et z pour la profondeur.

    2.1.3. Ajouter un puits

    Pour ajouter un puits, il faut aller dans le menu "Ajouter un puits".
    Ensuite, il suffit juste de donner le nom du puits pour l'ajouter dans la base de données.


    2.2. Suppression de données

    2.2.1. Supprimer une sonde

    Pour supprimer une sonde, il faut aller dans le menu "Supprimer une sonde".
    Ensuite, il suffit de sélectionner LA OU LES sondes que l'on veut supprimer de la base de données, et de valider en cliquant sur "Supprimer".
    Pour (dé)sélectionner plusieurs sondes, maintenez la touche CTRL + clic de la souris sur les sondes voulues.

    2.2.2. Supprimer une corbeille

    Pour supprimer une corbeille, il faut aller dans le menu "Supprimer une corbeille".
    Ensuite, il suffit de sélectionner LA OU LES corbeilles que l'on veut supprimer de la base de données, et de valider en cliquant sur "Supprimer".
    Pour (dé)sélectionner plusieurs corbeilles, maintenez la touche CTRL + clic de la souris sur les corbeilles voulues.
    ATTENTION, LA SUPPRESSION DE LA CORBEILLE NE SUPPRIME PAS LES SONDES QUI Y SONT ATTACHEES.

    2.2.3. Supprimer un puits

    Pour supprimer un puits, il faut aller dans le menu "Supprimer un puits".
    Ensuite, il suffit de sélectionner LE OU LES puits que l'on veut supprimer de la base de données, et de valider en cliquant sur "Supprimer".
    Pour (dé)sélectionner plusieurs puits, maintenez la touche CTRL + clic de la souris sur les puits voulus.

    2.3. Importation / Exportation

    Pour accéder à l'importation et l'exportation, il faut cliquer sur le menu Importation/Exportation.

    Importation

    Pour importer une base SQL, il suffit de parcourir les fichiers où est la base SQL, et ensuite de cliquer sur Importer.

    Exportation

    Pour l'exportation, il suffit d'entrer le nom que l'on veut donner à notre fichier puis de cliquer sur Exporter

    2.4. Arduino

    Pour accéder à l'Arduino, il faut cliquer sur le menu Arduino.
    Ensuite, il suffit d'indiquer la valeur de la fréquence que l'on veut, en millisecondes.
